# 🤖 Social Media Automation Suite

A Python-based browser automation project developed using **Selenium WebDriver** to automate login workflows across multiple social media platforms. The project demonstrates browser automation, web element interaction, object-oriented programming, and Python scripting for repetitive web tasks.

---

## 📖 Project Overview

The objective of this project is to explore browser automation by implementing automated login workflows for popular social media platforms. Each platform is implemented as an independent automation module, allowing reusable and maintainable code while demonstrating practical Selenium automation techniques.

---

## ✨ Features

- Automated login using Selenium WebDriver
- Browser automation with Chrome and Firefox
- Object-oriented implementation for each platform
- Automated web element interaction
- Keyboard event automation
- Cross-platform automation scripts
- Simple and reusable project structure

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/social-media-automation.git

cd social-media-automation
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

**Windows**

```bash
.venv\Scripts\activate
```

**Linux / macOS**

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install selenium
```

### Install Browser Driver

Download the appropriate browser driver:

- ChromeDriver (Google Chrome)
- GeckoDriver (Mozilla Firefox)

Ensure the driver executable is available in your system PATH or project directory.
